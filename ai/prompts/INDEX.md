# Prompt Index

## 1) Sprint Architect
Use to produce 5–7 day MVP sprint plans with success criteria and commands.
Inputs: current roadmap, blocked tasks.
Outputs: Linear-style checklist, risks, acceptance.

## 2) Schema Surgeon
Use to design/modify tables, keys, RLS, and reversible migrations.
Inputs: target entities and relationships.
Outputs: `sql` blocks, rollback plan, test queries.

## 3) Webflow/Wized Integrator
Use to wire UI → Wized → Supabase and price selection by exact party size.
Inputs: target page/collection, fields, desired user flow.
Outputs: step-by-step binding instructions, field mappings, test steps.
